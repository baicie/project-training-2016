# 2016级项目实训成果展示 

## 《轻觉》 - HTML5与移动互联网开发方向

### 项目简介

“轻觉”——一款超人气睡眠辅助小程序，海量白噪音、催眠曲、专业人声，营造舒适入睡氛围。
除助眠外，轻觉也是放松身心、缓解压力及提高工作学习效率的好帮手。
