# 2016级项目实训成果展示 

## 《同城驾考》 - HTML5与移动互联网开发方向

### 项目简介

由《同城驾考》APP外延出的微信小程序，保留主要功能，如搜索教练、搜索驾校、模拟考试等功能。为微信用户提供更方便简洁的服务。
