# 2016级项目实训成果展示 

## 《玩转周末》 - HTML5与移动互联网开发方向

### 项目简介

一款“生活服务类App”——寻求灵感，过充实的周末！

你有好好为自己定制一个完整的周末计划吗？你的周末过得充实而有意义吗？玩转周末APP，就是为想过有趣周末的当代青年提供灵感，比如看一场痛快淋漓的电影、有一个短期的小旅途或者是让自己的胃得到充分的满足等等，让自己的周末充实起来，从而拥有一个好心情，有利于下周的工作能够高效率的完成。同时让他们能够合理的安排自己的周末计划，使周末生活过的更加有意义、丰富多彩！
