# 2016级项目实训成果展示 

## 《游迹+》 -  Java与移动智能设备开发方向

###  项目简介

一款“旅游App”！

为那些喜欢旅行的人们打造一个畅想言论，分享旅行体验的社区，让旅行生活匾额丰富多彩，不在“耳听为虚”，做到真正的“眼见为实；为需要旅行信息的群体提供实时的，详细的，人性化的旅行信息，让用户可以花费更少的时间在收集信息上。
