# 2016级项目实训成果展示 

## 《灵跃》 -  Java与移动智能设备开发方向

###  项目简介

**灵跃（英文名：Linear）**是一款运行在 Android 端的运动类 app，它可以为喜欢健身活动和有健身意向的人群提供一个集训练、健康、课程和社交为一体的综合平台。用户可以通过 app 开展训练（如跑步、健身操）、通过分析健康指标和环境数据得到合理的健康建议、通过安排日程制定运动计划，并有社交和分享功能，app 还有线上课程和运动知识供用户学习。
