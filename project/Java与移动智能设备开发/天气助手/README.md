# 2016级项目实训成果展示 

## 《天气助手》 -  Java与移动智能设备开发方向

###  项目简介

为了方便人们的生活出现，制作了一个简洁明了的七天天气查询，可以实现多地的天气查询、空气质量查询等，并根据天气和气温给出穿衣建议和生活建议
