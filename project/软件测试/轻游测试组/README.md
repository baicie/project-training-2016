# 2016级项目实训成果展示 

## 《轻游测试组》 - 软件测试方向

###  项目简介

轻游是一款“旅游攻略型App”——衣食住行，样样行！

人的一生，在工作学习生活中会面临各式各样的烦恼，忧虑，纠结还有各种各样的问题。越是这种时候，我们更要放松自己，给自已一段无忧无虑的时光，让自己彻底远离纷扰与喧嚣。更可以寻找到志同道合的朋友，多方面满足自己、提升自己。

针对于app的四大模块及附属子模块进行功能和性能测试。
